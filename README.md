# siem_close_incidents
A Go-based command-line tool for managing SIEM incidents via Positive Technologies Max Patrol official API

## Features
- **Get incidents**: Fetch and display incident details
- **Update incidents**: Change incident status and assignees
- **Filter by correlation name**: Target specific incident types
- **Batch processing**: Handle multiple incidents at once

## Build from source
1. Pull soruce files
2. Run `go build -o siem-tool main.go`

## Installation
1. Download latest release
2. Place where you want
3. Run as binary file
